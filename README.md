# OtterLink

个人工具项目合集。仓库由若干相互独立的子项目组成，每个子项目一个顶层目录，自带文档、依赖与部署说明，可单独使用和演进。

## 子项目

| 目录 | 说明 | 文档 |
| --- | --- | --- |
| [bjx_power/](bjx_power/) | 北极星电力网（bjx.com.cn）每日资讯自动抓取与展示系统：工作日定时抓取 20 个栏目的 24 小时增量文章，生成每日简报并以静态网站展示（生产：<https://bjx.geekbit.org>） | [bjx_power/README.md](bjx_power/README.md) |

新增子项目时：创建顶层目录并在上表登记一行，子项目内部保持自包含（独立的 README、依赖清单与部署文档）。
